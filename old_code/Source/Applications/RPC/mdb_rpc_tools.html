Display MetDB Program number status files (on PROD):

This utility will print details of all MetDB multi-user RPC program number status files on the IBM unix partition. The list contains program number, status (0=free), date of last use (YYYYMMDDhhmmss). Note: A status of 1 does not necessarily mean the program number is currently in use. It may have terminated abnormally, without being freed, e.g. an IBM crash. A server that has a "date of last use" greater than 6 hours is available for re-use, should all program number status flags be 1. 
200001 - 200020  MetDB test servers

200021 - 200100  Servers for general use

200101 - 200120  Servers for operational extraction use only
