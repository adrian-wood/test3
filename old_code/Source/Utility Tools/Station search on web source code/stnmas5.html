MetDB Station Details

Note that the Station name search will search for the string entered within any name on the Station Master.

For ICAO and WMO searches a partial entry eg 037 or EG will produce results containing all stations matching the search. Clicking on links within the results page will give full details for that station off the Full Station Master list.

For Decimal Latitudes and Longitudes a negative value indicates South and West respectively.
